# Cossack Earthworks Website

A simple one-page landing page for Cossack Earthworks, a professional excavation and site preparation company.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **React 18**

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Project Structure

- `app/page.tsx` - Main landing page
- `app/layout.tsx` - Root layout with metadata
- `app/globals.css` - Global styles and section styling
